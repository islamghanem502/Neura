# 🧬 Neura Health Systems

> **Enterprise medical infrastructure designed for privacy, speed, and elegance.**

Neura is a smart healthcare platform connecting patients with trusted medical services. It is designed to modernize how clinics operate by providing tools like AI-powered scribing, real-time sync, and personalized patient Digital Twins.

## 🚀 Features

- **Personal Digital Twin**: Track patient health progress, understand symptoms, and receive personalized care recommendations via an interactive virtual interface.
- **AI Documentation ("Zero Paperwork")**: Built-in LLM integrations convert doctor voice notes instantly into structured medical reports (SOAP notes).
- **Elite Specialist Network**: Easily discover and book appointments with world-class specialists using dynamic scheduling dashboards.
- **Enterprise Security Features**: HIPAA-compliant encrypted flows ensuring complete continuity of care between physicians, nurses, and pharmacies.
- **Premium UI/UX System**: Extremely polished, modern aesthetic leveraging custom CSS glassmorphism, fluid `reveal-on-scroll` micro-animations, layout grid overlays, and seamless routing. 

## 🛠️ Tech Stack

This project leverages a modern, highly performant frontend ecosystem:

- **Core**: [React 19](https://react.dev/) + [Vite 7](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Routing**: [React Router v7](https://reactrouter.com/)
- **Data Fetching**: [TanStack Query](https://tanstack.com/query/latest)
- **Forms**: [React Hook Form](https://react-hook-form.com/)
- **Icons & Graphics**: [Lucide React](https://lucide.dev/), [Three.js](https://threejs.org/), and [@tsparticles](https://particles.js.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🏎️ Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository and navigate into the project directory:
   ```bash
   cd Neura
   ```
2. Install the application dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server using Vite:

```bash
npm run dev
```

The app will be served locally (usually at `http://localhost:5173`).
